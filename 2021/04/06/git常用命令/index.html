git常用命令

Created2021-04-06|Updated2021-04-27|Git
|Word count:269|Reading time:1min|Post View:
  • git diif：查看修改的内容
  • git show：显示上次提交修改的内容
  • git log：显示提交日志
  • git status：查看当前分支状态
  • git add -p：挨个询问是否stage该内容变化（y：yes，n：no，s：把单个变化切分开……）
  • git checkout . ：退回修改之前的操作（add之前）
  • git reset . ：如果add了，用这个命令可以退回到add之前的状态，就可以使用checkout
  • git reset –hard head：如果commit想要撤回，使用这个操作（head：最近一个提交的log,有几个就代表上几个log,或者是+数字，比如~2的意思是上两个版本，可以使用版本号的前几个字符退回到指定的版本）
  • git reflog：可以查看HEAD移动情况，如果上一个命令移动了头节点，可以使用这个命令查看移动情况，查询到以前的版本号头部字符，可以使用上一个命令指向想要恢复的版本\
  • git pull –rebase：更新代码解决冲突，然后git rebase –continue
